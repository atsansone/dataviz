Clean Data

Data Is...

Messy

Data will almost never be ready to use. Someone had to enter in the data at some point; mistakes were made and inconsistencies are present.

Unpredictable

Content can and will change. Never expect the data to be in the same format each time. Or in one file.

Limited

Data is not all encompassing; it covers one aspect that the data provider wants. It may be degrees conferred, but not majors. It may be alumni location, but only by state. Sometimes you will need to dig. Sometimes you need to combine more than one source.

Working with Data

Keep Copies

Never Work on Originals

Data can be revised at a later date. You may make a mistake with the data and need to start over. Keep original files for archival and protective purposes.

Know Everything

Review the data for purpose. If there's a flag in column 206 that means "ignore record", best to know that.

Tools

UNIX Philosophy

Write programs that do one thing and do it well. Write programs to work together. Write programs to handle text streams, because that is a universal interface.
Use tools that do single, simple things and chain them together. Allows you to swap out parts later if needed.
